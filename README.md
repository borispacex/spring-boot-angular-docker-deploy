# SPRING BOOT 
Deploy proyecto java con angular

## BACKEND
Variables de entorno:
- DATABASE_URL
- DATABASE_USERNAME
- DATABASE_PASSWORD

Generar jar
```
./mvnw clean package -DskipTests
mvn clean package -DskipTests
```
Ir a la ruta
- http://localhost:8080/

## FRONTEND
Ir a la ruta
- http://localhost:4200/

## Comandos
```
docker-compose up
```
